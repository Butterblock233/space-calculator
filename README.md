# space-calculator
## 使用python实现的简易的立体几何计算器

## 用法

```bash
git clone https://github.com/virtualguard101/space-calculator.git
cd space-calculator
python -i main.py
```

### 在python交互环境中查看用法

```python
>>> help()
```
